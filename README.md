# @Growpro App Alquiler de Bicicletas

## Instrucciones

Correr el ambiente de desarrollo:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Abrira por defecto en [http://localhost:3000](http://localhost:3000).
